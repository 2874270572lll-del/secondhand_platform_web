🛠 二手交易平台开发流程（Vue3 + Element Plus + Spring Boot）
一、需求分析阶段（第 1 阶段）
1. 明确系统目标

本系统面向在校学生，提供一个安全、便捷的校园二手交易平台，实现商品发布、浏览、搜索及交易等功能。

2. 确定用户角色

普通用户（买家 / 卖家）

管理员（可选）

3. 功能需求分析

用户注册与登录

商品发布、编辑、删除

商品分类与搜索

商品详情查看

下单与订单管理

收藏商品

后台商品审核（选做）

二、系统总体设计阶段（第 2 阶段）
1. 系统架构设计

采用 前后端分离架构：

Vue3 + Element Plus（前端）
        ↓ REST API
Spring Boot（后端）
        ↓
      MySQL

2. 技术选型
层级	技术
前端	Vue3、Element Plus、Axios
后端	Spring Boot、Spring MVC
数据层	MyBatis / JPA、MySQL
安全	JWT
工具	GitHub、Swagger
3. 数据库设计

设计用户表、商品表、订单表等

绘制 ER 图

明确表之间的关系
